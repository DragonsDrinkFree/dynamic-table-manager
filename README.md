# Overview
This module provides an improved interface for creating/managing tables within Foundry.
Table creation can be done manually, by copy & Past, or by selecting a local PDF and scanning tables directly from it. Multiple tables can be made simultaneously. Multi-column tables can be copy/pasted and automatically split into separate tables with a compound table created to roll on each table to produce a result.

Ordering of results can be easily done by changing a number or dragging results where desired. Weight has become a hidden stat, favoring natural ttrpg language of ranges "1-2, 3-4,5-6".

Compendiums can be scanned for key words to automatically replace text tables with linked objects for Actors/Items/journals/tables.

Document quick search: Can quickly search from within the table editor for other objects to provide quick updates to linking. Very easy for linking sub-tables.

## Additional Features & Bug Reports
